Most of features are available on the toolbars and in the menus on top of a main window.

Level

  • The Section settings window allows you to change the properties of the level's sections, such as background, size and music.
  • The Tileset itembox contains items you can place in your levels. Right-click items to mark them as favorites for easy access.
  • The Warps and Doors window can be used to place warp boxes that players can use to move between sections and levels.
  • Layers are tag groups to move, show or hide groups of objects using events or lunalua. For example, they can be used to create moving pipes.
  • Events are configurable scenarios to manipulate layers, sections or player movement. They can be triggered from block and npc objects or from lunalua.
  • The Search box can help you find items in the level.
  • The Level settings window can be used to configure a level timer or change the character players start with.
  • Using the Tileset itembox's custom tab, you can create your own tilesets with items you use frequently.

World map

  • The World settings window allows you to configure episode settings.
  • The Tileset itembox contains items you can place on your world map. Right-click items to mark them as favorites for easy access.
  • The Search box can help you find items on the map.
Use the context menu to get access to properties of target items or selected item groups.

Don't know what the context menu is? The context menu is a menu which pop-ups when you press the right (left for left-handed) mouse button.
When you can't find the used section or a zone on a huge map, you can quickly return back to top-left or bottom-left of a level section or to the 0-0 position on the world map.
Position bookmarks are editor tools that don't affect the game. Place them in locations where you think you will need your editor camera to look at frequently.
Do you remember the bug with placing vines in the SMBX 1.3 editor? You can clean-up your level from junk NPCs caused by this bug by the menu item:
Tools ->Custom data ->Remove NPC garbage...
You can share your ideas and creations publicly by exporting your level or world map parts as image files.
To export part of a level or world map screen you can press the F12 key or select the File -> Export to image menu item.
Note: To export entire level sections you need to select the File -> Export section to image... menu item.
You can use the middle mouse button (press scrolling wheel) for advanced interactions.
  • If there are no selected items, hand-drag mode will be enabled.
  • If you select one item, you will start placing new items of the same type.
  • If you select multiple items, they will be duplicated to the current mouse position on the top-left corner of the selection area
There is an Overwrite mode which allows you to place new elements over existing ones, overriding them. An icon will appear on the toolbar when you pick up an item to place.
You can also place overlapping items by holding CTRL while placing.
Did you know that you can move Layers and Events through the list? Drag items in the list to change their order!
Did you know that you can convert legacy GIF+mask graphics in your levels/episodes into PNGs with full transparency and semi-transparency?
You can use GIFs2PNG to process that: Tools ->External tools ->Masked GIFs to PNG converter...
You have placement tools which you can find on the toolbar:
  • Line allows you to place a bunch of items by drawing a line. Useful for drawing columns, slopes, bridges, walls, gates, etc.
  • Rectangle allows you to place a bunch of items by drawing rectangles.
  • Circle allows placement bunch of items by drawing ovals.
  • Flood-fill allows filling of closed zones. Don't worry if you fill outside space, you can undo your mistake with the Ctrl+Z key combo
Note: Those buttons will be available when you pick up an item to place.
If you want to place elements more accurately, you can zoom in and change grid alignment by using the Snap to grid menu.
Scroll your mouse wheel while holding the CTRL key to zoom in or out.
